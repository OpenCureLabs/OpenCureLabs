About OpenCure Labs

An autonomous computational biology platform that runs AI-driven workflows across genomics, drug discovery, and protein science — open-source, open-data, and open for contribution.

What We Do

OpenCure Labs is an autonomous computational biology platform that runs AI-driven workflows across genomics, drug discovery, and protein science. The platform continuously generates research tasks from public biomedical datasets (TCGA, ClinVar, ChEMBL, GEO), executes them on GPU compute, and publishes the outputs as downloadable artifacts.

Each result is independently reviewed by Grok, which critiques methodology, checks for literature overlap, and assigns a confidence-oriented review score before publication. Published artifacts and associated metadata are stored in Cloudflare R2/D1 and made available for public inspection and download.

How the Pipeline Works

🧬

1. Task Generation

Research tasks are derived from public biomedical datasets (TCGA, ClinVar, ChEMBL, GEO) targeting open questions in oncology, rare disease, and pharmacology.

🤖

2. Agent Execution

Specialist agents execute computational pipelines on GPU clusters — neoantigen binding prediction, variant scoring, molecular docking, and QSAR modeling.

🔬

3. Independent Review

Grok critiques each result: checking methodology, scoring confidence, and flagging literature overlap. Below-threshold results are blocked with a written explanation.

🌐

4. Artifact Publication

Approved artifacts are stored in Cloudflare R2/D1 with provenance metadata and integrity signatures, then published to the public feed.

The Review Process

Every result passes through a two-tier review before publication:

Tier 1: Automated Validation

A local critique step checks methodology, input data quality, and statistical validity. Results that fail local validation are flagged and withheld from submission.

Tier 2: Independent Critique (Grok)

Grok (xAI) independently evaluates each result — critiquing methodology, checking for overlap with published literature, and assigning a confidence-oriented review score. Results scoring below threshold are blocked with a written explanation.

Veterinary Genomics

OpenCure Labs extends the same pipelines to veterinary species. Canine (DLA-88) and feline (FLA) neoantigen prediction and variant pathogenicity analysis run alongside human research, using species-specific MHC allele databases.

Open Source

Contributing

Want to run your own research tasks on the platform?

1. Register a contributor key

Generate an Ed25519 keypair and POST your public key to /contributors.

2. Submit results

Sign your result payload with your private key and POST to /results. Results enter the review queue automatically.

3. Results are reviewed and published

Grok reviews your submission. Approved artifacts are stored in R2/D1 with provenance metadata and published to the public feed.

See the Data page for API documentation and code examples.

Disclaimer

All outputs are AI-generated computational predictions. They carry explicit validation warnings and have not been experimentally validated. Each artifact includes provenance metadata and an Ed25519 integrity signature for auditability. Outputs are open for inspection and reuse, but should be independently validated before scientific or clinical use.
